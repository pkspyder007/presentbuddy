# Icons

This directory should contain app icons for all platforms:
- `icon.png` - Linux icon (512x512 PNG)
- `icon.icns` - macOS icon (multi-resolution ICNS)
- `icon.ico` - Windows icon (multi-resolution ICO)

For now, these are placeholders. Generate proper icons using:
- Online tools like https://www.icoconverter.com/
- Or use electron-icon-maker
